How to use ?

If you want to play single mode, 
complie oos.c or ooc.c then excute oos or ooc
then select "혼자하기" then you can play game with computer.

If you want to play multi mode, 
First command ./oos to make server file
Then make another shell in another computer or another environment 
command ./ooc to make Client file

1. select "같이하기" at oos program  and oos will wait client 
2. select "같이하기" at ooc program and game will start

Then you can enjoy our program.
